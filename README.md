# RIPng Simulator (IPv6 Distance Vector Routing)

This project shows the RIPng routing protocol used in IPv6 networks.

## Features
- Distance Vector Routing Algorithm
- Hop Count Metric (Max = 16)
- Packet Exchange Animation
- Step-by-Step Simulation (Pause / Next Step)
- Before vs After Routing Tables
- Link Failure Simulation
- Router Selection (Click to view table)

## Technologies
- Java (Swing)
- GUI-based simulation

## How it Works
Each router exchanges routing tables with neighbors and updates its routes based on:
New Cost = Cost to Neighbor + Neighbor’s Cost

The simulation continues until convergence.
